HTML Tags And Their Use
TAG NAME DESCRIPTION / USE
<html> HTML This tag encloses all the HTML content of a webpage
<head> Head This tag the metadata of the webpage and it's title.
<body> Body This tag encloses the visible contents of the webpage.
<title> Title Contained in the head tag displays the title/name of the webpage on the webpage's tab.
<link> Link Also found within the head tag, this is used to link external files, including CSS files to the HTML file.
<script> Script This is used to exclusively used to write JavaScript code within the HTML document.
<style> Style Just like the script tag, this is used to write CSS code directly within the HTML document.
<header> Header It sections off the introduction of the webpage, may contain navigation links, logos, menus, etc.
<main> Main This tag sections of the main content of the webpage from its header and footer.
<nav> Navigation It contains the various navigation links.
<h1>-<h6> Title Used for the title of webpage or its content in the webpage. With h1 being the largest size importance followed by h2, then h3 through to h6.
<div> Division This tag is used to contain various divisions of content.
<span> Span This tag encloses other HTML elements including text, images, etc.
<form> Form For creating HTML forms.
<table> Table It is used to include tables into the HTML document
<p> Paragraph For enclosing text
<a> Anchor It is used to create links to webpages.
<img> Image It is used to display images.
<audio> Audio This contains audio files
<video> Video As the name implies, it is used to display videos.
<embed> Embed This tag encloses all media contents, of a webpage, audio images, videos, gifs etc.
<footer> Footer This tag encloses the closing part of the webpage. It usually contains the author's contact, Copyright information etc.
<address> Address Contains the contact details of the developer, eg, name, address, email etc.

HTML Tag Attributes AND Their Uses
Attribute Name Tag(s) Use
href Hypertext Reference a, link For directing the location within the computer or on the web where the file is found.
rel Relationship link Used when linking CSS files. It determines the relationship between the HTML document and the CSS file.
src Source img, video, audio, embed Also used to direct the HTML document as to where the media file is located. It could be on the desktop or on the internet.
class Class Any tag within the body element. It is used as a special identity for an element, especially if there are several of such elements and we want to target the element specifically.
id Identity Any tag within the body element. Like the class attribute, it is also used to target specific elements, however it is more specific than the class attribute.
alt Alternative img Specifies what text the browser should display when the image doesn't load.
