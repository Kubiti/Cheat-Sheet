Introduction

HTML and CSS are the building blocks of every webpage you see on the internet. Thus, to become a web developer you need to know these two languages. In this page, I cover the basics of the two languages. You can use this as a note or a reference to help you on your web development journey.

HTML Tags

All HTML contents are enclosed in tags; text, images, videos etc. are put in tags. The content is put between opening and closing tags. The content together with the opening and closing tags form an HTML element. Below is a paragraph element which displays text:
<p>This is a paragraph</p>
As seen above, the text 'This is a paragraph' begins with an opening tag, which is made of the tag name (p) enclosed in a less than and greater than sign. It is then ended in a closing tag, which has a forward slash after the less than sign. However, not all content have closing tags. Some tags are self-closing and hence do not need a closing tag. An example is the image (img) tag which is used show images. It is shown below:
<img src="image.png">
As seen above, there is no need for a closing tag. Another way of representing self-closing tags is <img>
Did you notice the src="image.png", this is an attribute and you can learn more about them in the attribute section. The tables below shows some basic HTML tags and their uses.

HTML Tag Attributes

HTML attributes are keywords that are put in opening tags to modify the behavior of the element. In our HTML tags section we saw an example of an attribute:
<img src="image.png">
here src (source) is the attribute, and it is used to specify location of the image file. This can be a folder within your desktop, or a location on the internet. Below is a table showing some HTML tag attributes and their use.

HTML TAG ATTRIBUTES AND THEIR USES
Attribute Name Tag(s) Use
href Hypertext Reference a, link For directing the location within the computer or on the web where the file is found.
rel Relationship link Used when lining CSS files. It determines the relationship between the HTML document and the CSS file. "Stylesheet" is normally used as the value.
src Source img, video, audio, embed Also used to direct the HTML document as to where the media file is located. It could be on the desktop or on the internet.
class Class Any tag within the body element. It is used as a special identity for an element, especially if there are several of such elements and we want to target the element specifically.
id Identity Any tag within the body element. Like the class attribute, it is also used to target specific elemnts, however it is more specific than the class attribute.
href Hypertext Reference a, link For directing the location within the computer or on the web where the file is found.
alt Alternative img Specifies what text the browser should display when the image doesn't load.
